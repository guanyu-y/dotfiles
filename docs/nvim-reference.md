# Neovim キーバインドリファレンス

リーダーキー: `,`

## 基本操作

| キー | 動作                     |
| ---- | ------------------------ |
| `U`  | Redo (やり直し)          |
| `v`  | ビジュアルブロックモード |
| `zj` | カーソル行を画面上部に   |
| `zk` | カーソル行を画面下部に   |

## 移動

| キー | 動作       |
| ---- | ---------- |
| `H`  | 10文字左へ |
| `J`  | 10行下へ   |
| `K`  | 10行上へ   |
| `L`  | 10文字右へ |

## タブ操作

| キー | 動作             |
| ---- | ---------------- |
| `tn` | 新しいタブを開く |
| `tq` | タブを閉じる     |
| `tl` | 次のタブへ       |
| `th` | 前のタブへ       |

## ウィンドウ分割・移動

| キー | 動作               |
| ---- | ------------------ |
| `,s` | 横分割             |
| `,v` | 縦分割             |
| `,,` | ウィンドウを閉じる |
| `gh` | 左のペインへ移動   |
| `gj` | 下のペインへ移動   |
| `gk` | 上のペインへ移動   |
| `gl` | 右のペインへ移動   |

## クリップボード

| キー  | モード        | 動作                           |
| ----- | ------------- | ------------------------------ |
| `,y`  | Normal/Visual | システムクリップボードにコピー |
| `,Y`  | Normal        | カーソル位置から行末までコピー |
| `,yy` | Normal        | 行全体をコピー                 |
| `,p`  | Normal/Visual | クリップボードから貼り付け     |
| `,P`  | Normal/Visual | カーソル前に貼り付け           |

## コメント

| キー | モード        | 動作             |
| ---- | ------------- | ---------------- |
| `,/` | Normal/Visual | コメントをトグル |

## NvimTree (ファイルツリー)

### 起動・終了

| キー     | 動作                  |
| -------- | --------------------- |
| `Ctrl-t` | NvimTreeを開く/閉じる |
| `q`      | NvimTreeを閉じる      |

### ファイル操作

| キー          | 動作                         |
| ------------- | ---------------------------- |
| `Enter` / `o` | ファイルを開く               |
| `Tab`         | プレビュー                   |
| `a`           | 新規作成 (ファイル/フォルダ) |
| `r`           | 名前変更                     |
| `d`           | 削除                         |
| `x`           | 切り取り                     |
| `c`           | コピー                       |
| `p`           | 貼り付け                     |
| `y`           | ファイル名をコピー           |
| `Y`           | 相対パスをコピー             |
| `gy`          | 絶対パスをコピー             |

### ナビゲーション

| キー | 動作                         |
| ---- | ---------------------------- |
| `-`  | 親ディレクトリへ移動         |
| `BS` | フォルダを閉じる             |
| `P`  | 親ディレクトリへカーソル移動 |
| `J`  | 最後の兄弟へ                 |
| `K`  | 最初の兄弟へ                 |
| `>`  | 次の兄弟へ                   |
| `<`  | 前の兄弟へ                   |

### 開き方

| キー    | 動作                         |
| ------- | ---------------------------- |
| `Enter` | カレントウィンドウで開く     |
| `s`     | システムで開く               |
| `O`     | ウィンドウピッカーなしで開く |

### 分割して開く

| キー           | 動作             |
| -------------- | ---------------- |
| (メニューから) | 縦分割で開く     |
| (メニューから) | 横分割で開く     |
| (メニューから) | 新しいタブで開く |

### 表示切替

| キー | 動作                             |
| ---- | -------------------------------- |
| `H`  | 隠しファイル (dotfiles) 表示切替 |
| `I`  | .gitignoreファイル表示切替       |
| `B`  | バッファにないファイル表示切替   |
| `U`  | カスタムフィルター表示切替       |
| `C`  | Git cleanファイルのみ表示切替    |
| `E`  | 全て展開                         |
| `W`  | 全て折りたたむ                   |
| `R`  | リフレッシュ                     |

### フィルター・検索

| キー | 動作           |
| ---- | -------------- |
| `f`  | フィルター開始 |
| `F`  | フィルター解除 |
| `S`  | ノード検索     |

### Git操作

| キー | 動作                       |
| ---- | -------------------------- |
| `ga` | git add (ステージング)     |
| `gu` | git unstage (アンステージ) |
| `[c` | 前のGit変更へ              |
| `]c` | 次のGit変更へ              |

### Telescope連携

| キー     | 動作                   |
| -------- | ---------------------- |
| `Ctrl-f` | ファイル検索           |
| `Ctrl-g` | テキスト検索 (grep)    |
| `Ctrl-d` | ディレクトリにジャンプ |
| `Space`  | コマンドメニュー表示   |

### その他

| キー | 動作                 |
| ---- | -------------------- |
| `.`  | コマンド実行         |
| `g?` | ヘルプ表示           |
| `m`  | ブックマークをトグル |

## Telescope (ファジーファインダー)

| キー     | 動作                     |
| -------- | ------------------------ |
| `Ctrl-f` | ファイル検索             |
| `Ctrl-g` | テキスト検索 (live grep) |

### Telescope内の操作

| キー                | 動作             |
| ------------------- | ---------------- |
| `Ctrl-n` / `Ctrl-p` | 次/前の候補      |
| `Enter`             | 選択して開く     |
| `Ctrl-x`            | 横分割で開く     |
| `Ctrl-v`            | 縦分割で開く     |
| `Ctrl-t`            | 新しいタブで開く |
| `Esc`               | 閉じる           |
