# tmux キーバインドリファレンス

Prefixキー: `Ctrl-j`

> 以下、`Prefix` は `Ctrl-j` を押してから次のキーを押すことを意味します

---

## セッション管理

| キー               | 動作                       |
| ------------------ | -------------------------- |
| `tmux`             | 新しいセッションを開始     |
| `tmux new -s 名前` | 名前付きセッションを開始   |
| `tmux ls`          | セッション一覧             |
| `tmux a`           | 最後のセッションにアタッチ |
| `tmux a -t 名前`   | 指定セッションにアタッチ   |
| `Prefix + d`       | セッションからデタッチ     |

---

## ウィンドウ操作

| キー              | 動作                   |
| ----------------- | ---------------------- |
| `Prefix + c`      | 新しいウィンドウを作成 |
| `Prefix + ,`      | ウィンドウ名を変更     |
| `Prefix + &`      | ウィンドウを閉じる     |
| `Prefix + w`      | ウィンドウ一覧         |
| `Prefix + n`      | 次のウィンドウへ       |
| `Prefix + p`      | 前のウィンドウへ       |
| `Prefix + 数字`   | 指定番号のウィンドウへ |
| `Prefix + Ctrl-h` | 前のウィンドウへ       |
| `Prefix + Ctrl-l` | 次のウィンドウへ       |

---

## ペイン分割

| キー         | 動作                  |
| ------------ | --------------------- |
| `Prefix + s` | 横分割 (上下に分ける) |
| `Prefix + v` | 縦分割 (左右に分ける) |

---

## ペイン移動

| キー           | 動作                       |
| -------------- | -------------------------- |
| `Prefix + h`   | 左のペインへ               |
| `Prefix + j`   | 下のペインへ               |
| `Prefix + k`   | 上のペインへ               |
| `Prefix + l`   | 右のペインへ               |
| `Alt + ←`      | 左のペインへ (Prefixなし)  |
| `Alt + →`      | 右のペインへ (Prefixなし)  |
| `Alt + ↑`      | 上のペインへ (Prefixなし)  |
| `Alt + ↓`      | 下のペインへ (Prefixなし)  |
| `Ctrl-o`       | 次のペインへローテーション |
| `Ctrl-Shift-o` | 前のペインへローテーション |

---

## ペインリサイズ

| キー         | 動作          |
| ------------ | ------------- |
| `Prefix + H` | 左に5単位拡大 |
| `Prefix + J` | 下に拡大      |
| `Prefix + K` | 上に拡大      |
| `Prefix + L` | 右に5単位拡大 |

---

## ペイン操作

| キー             | 動作                           |
| ---------------- | ------------------------------ |
| `Prefix + x`     | ペインを閉じる                 |
| `Prefix + z`     | ペインを最大化/元に戻す        |
| `Prefix + !`     | ペインを新しいウィンドウに移動 |
| `Prefix + q`     | ペイン番号を表示               |
| `Prefix + {`     | ペインを前に移動               |
| `Prefix + }`     | ペインを後に移動               |
| `Prefix + Space` | レイアウトを切り替え           |

---

## コピーモード (Vim風)

| キー         | 動作             |
| ------------ | ---------------- |
| `Prefix + [` | コピーモード開始 |
| `v`          | 選択開始         |
| `Ctrl-v`     | 矩形選択         |
| `y`          | コピー           |
| `q` / `Esc`  | コピーモード終了 |
| `Prefix + ]` | 貼り付け         |

### コピーモード中の移動

| キー      | 動作            |
| --------- | --------------- |
| `h/j/k/l` | 左/下/上/右     |
| `w/b`     | 次/前の単語     |
| `0/$`     | 行頭/行末       |
| `g/G`     | 先頭/末尾       |
| `/`       | 検索            |
| `n/N`     | 次/前の検索結果 |

---

## 設定リロード

| キー         | 動作                 |
| ------------ | -------------------- |
| `Prefix + r` | 設定ファイルを再読込 |

---

## ステータスバー表示

現在のステータスバーに表示される情報:

| 位置 | 内容                                   |
| ---- | -------------------------------------- |
| 左   | セッション名                           |
| 中央 | ウィンドウ一覧                         |
| 右   | 日付、時刻、バッテリー残量、ローカルIP |

---

## よく使うコマンド

```bash
# セッション
tmux new -s work          # "work"という名前でセッション作成
tmux kill-session -t work # セッション削除
tmux rename-session 新名前 # 現在のセッション名を変更

# ウィンドウ
tmux new-window -n 名前   # 名前付きウィンドウ作成

# その他
tmux list-keys            # キーバインド一覧
tmux info                 # 情報表示
```
